# FE UI/UX 디자인 가이드

이 문서는 Memecore 프론트엔드(FE) 프로젝트의 UI 디자인 및 스타일링 작업을 위한 안내서입니다.

이 프로젝트는 UI 디자인 작업을 위해 React와 같은 복잡한 개발 환경 설정 없이, 순수 HTML과 CSS만으로 작업할 수 있는 환경을 제공합니다.

---

## **디렉토리 구조 안내**

프론트엔드(`fe`) 폴더의 주요 구조와 각 역할은 다음과 같습니다.

-   **`fe/style-guide.html`**
    UI/UX 디자이너가 가장 중요하게 봐야 할 파일입니다. 이 파일은 프로젝트의 모든 UI 컴포넌트(버튼, 카드, 내비게이션 바 등)의 **HTML 구조를 담고 있는 실시간 디자인 프리뷰**입니다. 디자인 작업을 할 때 이 파일을 웹 브라우저로 열어두고, CSS 수정 후 새로고침하며 시각적인 결과물을 확인합니다.

-   **`fe/src/styles/`**
    프로젝트의 모든 CSS 파일이 모여있는 곳입니다. UI/UX 디자이너는 주로 이 폴더 내의 파일들을 수정하게 됩니다.
    -   `main.css`: 모든 CSS 파일을 하나로 모으는 중심 파일입니다.
    -   `_variables.css`: 프로젝트의 공통 색상, 폰트 등을 변수로 관리합니다.
    -   `_base.css`: 웹사이트의 전역 스타일을 정의합니다.
    -   `components/`: 각 UI 조각들에 대한 개별 스타일 파일들이 있습니다.

-   **`fe/src/components/` & `fe/src/pages/`**
    React 컴포넌트 파일(`tsx`)이 위치하는 곳입니다. 개발자가 `style-guide.html`의 HTML 구조와 동일한 결과물을 만들도록 작업하는 공간이므로, 디자이너는 이 폴더를 신경쓰지 않아도 괜찮습니다.

---

## **화면 디자인 작업 방법**

화면 디자인 작업을 시작하고 싶으시다면, 먼저 **`fe/style-guide.html` 파일을 웹 브라우저로 열어보세요.**

그리고 `fe/src/styles/` 폴더에 있는 CSS 파일들을 수정하면서, 브라우저를 새로고침하시면 UI가 변경되는 것을 바로 확인하실 수 있습니다. 이 곳에서 자유롭게 상상력을 발휘하여 멋진 디자인을 만들어주시면 됩니다!
