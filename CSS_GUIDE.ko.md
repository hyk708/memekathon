# CSS 작업을 위한 개발 안내서

이 문서는 Memecore 프로젝트의 UI 디자인 및 CSS 스타일링 작업을 위한 안내서입니다. React나 다른 복잡한 설정 없이, CSS와 HTML만으로 디자인 작업을 진행할 수 있습니다.

---

## 1. 핵심 원칙: 폴더와 파일

- **당신의 작업 공간:** 모든 CSS 작업은 `fe/src/styles/` 폴더 안에서 이루어집니다. 이 폴더 외의 다른 파일은 수정할 필요가 없습니다.
- **실시간 미리보기:** 당신이 작성한 CSS의 결과물은 `fe/style-guide.html` 파일을 웹 브라우저에서 열어서 확인합니다.

---

## 2. 전체 작업 순서

1.  `fe/style-guide.html` 파일을 즐겨 사용하는 웹 브라우저(Chrome, Firefox 등)에서 엽니다.
2.  `fe/src/styles/` 폴더 내의 `.css` 파일들을 에디터로 엽니다.
3.  CSS 코드를 수정하고 저장합니다.
4.  열어둔 `style-guide.html` 브라우저를 **새로고침**하면, 방금 수정한 내용이 즉시 반영되어 보입니다.
5.  원하는 디자인이 나올 때까지 3-4번 과정을 반복합니다.

> **포인트:** React 개발 서버를 실행(`npm run dev`)할 필요가 전혀 없습니다. `style-guide.html` 파일만 있으면 모든 디자인 작업이 가능합니다.

---

## 3. 폴더 구조 살펴보기

당신은 아래 구조에서 `styles` 폴더만 신경 쓰면 됩니다.

```
/fe/src/
├── styles/          <- ✅ 당신의 기본 작업 공간
│   ├── main.css         (모든 CSS 파일을 불러오는 중심 파일)
│   ├── _variables.css   (색상, 폰트 등 공통 변수)
│   ├── _base.css        (body, h1 등 전역 스타일)
│   └── components/      (컴포넌트별 스타일)
│       ├── _navigation.css
│       ├── _buttons.css
│       └── _cards.css
│
└── ... (나머지는 React 개발자의 작업 공간)
```

---

## 4. 새로운 UI 추가 작업

만약 새로운 UI(예: 유저 프로필 카드) 디자인이 필요할 경우, 제가 먼저 `style-guide.html`에 해당 UI의 기본 HTML 뼈대를 추가하고 알려드릴 겁니다.

그러면 당신은 평소처럼 `style-guide.html`을 새로고침해서 새로 생긴 뼈대를 확인하고, `styles` 폴더에 CSS를 추가하여 디자인을 완성하면 됩니다.
